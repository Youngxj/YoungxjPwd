YoungxjPwd开源密码本使用说明：
首先YoungxjPwd是一款开源的密码本项目，如果你是付费购买，那么恭喜你，你被骗了。

首先你需要导入pwd.sql到你的数据库
然后你需要修改config.php中的数据库信息
之后访问/admin/进行测试账号是否正常登录


1、如何正常运行这款项目？
本项目采用监控执行任务的方式执行密码本用户注册，忘记密码，预约等所有邮件发送的操作，建议监控时长为1分钟，监控地址为http://localhost/admin/ajax.php?action=monitor&key=Youngxjpwd
，监控任务添加之前请完善后台设置中smtp信息配置。

2、用户管理中心自带的id为1的账号
此账号请勿删除，可以禁用，此账号是作为站长管理员的附属账号，添加的密码本，备忘录等信息均会附属给id为1的账号，当然如果该账号没有禁用也是可以单独使用。

3、默认的账号密码？
前台、后台默认账号都为admin，密码：admin000，请及时修改。
